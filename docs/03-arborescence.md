# 🗺️ Étape 3 — Définir l’arborescence du site

---

## 🧭 Objectif de l’arborescence

Définir la structure du site et les différentes pages qui composent l’expérience.

L’arborescence doit permettre :

- une navigation simple
- une exploration naturelle
- une progression dans la découverte de l’univers

---

## 🏗️ Structure globale du site

Le site est composé de plusieurs pages interconnectées.

### Pages principales

- Accueil (index)
- Journal
- Artefacts
- Carte
- Trace

---

## 📄 Détail des pages

### 🏠 1. Accueil

Rôle :

- introduire l’univers
- poser l’ambiance
- inviter à explorer

Contenu :

- hero immersif
- texte d’introduction
- accès aux autres pages

---

### 📜 2. Journal

Rôle :

- représenter les pensées ou traces de la conscience

Contenu :

- liste de “chapitres”
- textes courts ou fragments
- structure en articles

---

### 🧿 3. Artefacts

Rôle :

- montrer des éléments visuels du monde

Contenu :

- galerie d’images
- descriptions
- éléments mystérieux

---

### 🗺️ 4. Carte

Rôle :

- donner une représentation abstraite de l’espace

Contenu :

- zones ou sections
- éléments visuels
- navigation alternative

---

### ✍️ 5. Trace

Rôle :

- permettre à l’utilisateur de laisser une empreinte

Contenu :

- formulaire simple
- message
- nom (optionnel)

---

## 🔗 Navigation

Le site doit permettre :

- d’accéder facilement à toutes les pages
- de revenir à l’accueil
- de naviguer sans se perdre

La navigation doit être :

- simple
- discrète
- cohérente avec l’univers

---

## 🧠 Logique d’exploration

L’utilisateur doit pouvoir :

- commencer par l’accueil
- découvrir les pages dans n’importe quel ordre
- revenir en arrière facilement

Il n’y a pas de parcours imposé, mais une exploration libre.

---

## 💡 Hiérarchie des pages

Accueil  
→ Journal  
→ Artefacts  
→ Carte  
→ Trace

Toutes les pages sont importantes, mais :

- l’accueil est le point d’entrée
- le journal est le cœur narratif

---

## ✅ Validation de l’étape

L’étape est validée si :

- toutes les pages sont définies
- leur rôle est clair
- la navigation est pensée
- la structure globale est cohérente
